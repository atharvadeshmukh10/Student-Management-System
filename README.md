# Student-Management-System (C++)
A simple console-based **Student Management System** built using **C++**.
This program allows users to manage student records through a menu-driven interface.

---

## Features

* -Add new students
* -Display all student records
* -Search student by ID
* -Remove student from the system
* -Simple menu-driven interface

---

## How the Program Works

1. The program stores student records in an array of `Student` objects.
2. Each student contains:

   * **ID**
   * **Name**
   * **Age**
3. Users interact with the system using a **menu**.
4. The program performs operations such as **adding, displaying, searching, and removing students**.

---

## Example Menu

```
Student Management System

1. Add Student
2. Display Students
3. Search Student
4. Remove Student
5. Exit
```
